Training

Below, links to step-by-step tutorials for the CFD and FEA workbenches are provided, along with the required CAD geometry files. For the CFD analyses, the steps described in the tutorials can be reproduced by runnning the provided "FCMacro" script(s) in numerical order. The mesher and solver can then be run to produce the output shown in the tutorial.

The list below links to each tutorial and its supporting files. Note that the macro files are kept up to date with the latest developments in the workbench, and are contained in the 'demos' folder of the repository (download all). By contrast, the tutorials (download all) are not updated, and therefore there is some divergence between the two. However they do provide a general guideline of the steps to be followed, while the scripts can be run to produce an up to date demonstration of the capabilities of the workbench.
