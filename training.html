Training

Below, links to step-by-step tutorials for the CFD and FEA workbenches are provided, along with the required CAD geometry files. For the CFD analyses, the steps described in the tutorials can be reproduced by runnning the provided "FCMacro" script(s) in numerical order. In cases where the geometry is provided in a .fcstd file, this should be loaded first. The mesher and solver can then be run to produce the output shown in the tutorial.

The list below links to each tutorial and its supporting files. Alternatively, the entire repository can be downloaded.
